# 🏸 MatchCock

**MatchCock**은 대회를 선택한 뒤, 원하는 클럽과 팀을 빠르게 찾아  
**팀원별 시간표를 효율적으로 생성할 수 있는 웹 애플리케이션**입니다.

정렬·필터 기능과 반응형 UI를 통해  
대회 규모가 크더라도 필요한 정보에 빠르게 접근할 수 있도록 설계되었습니다.

---

## ✨ 주요 기능

- **대회 선택 기반 시간표 생성**
  - 특정 대회를 선택하여 해당 대회에 참여하는 클럽 및 팀 관리
  - 팀원별 일정 확인 및 시간표 구성

- **클럽 / 팀 빠른 검색**
  - 정렬(Sort) 및 필터(Filter) 기능 제공
  - 많은 클럽과 팀 중에서도 원하는 대상 즉시 탐색 가능

- **반응형 디자인**
  - 데스크톱, 태블릿, 모바일 환경 모두 대응
  - 화면 크기에 따라 UI 자동 최적화

- **효율적인 시간표 구성 흐름**
  - 불필요한 단계 최소화
  - 직관적인 UI로 빠른 일정 생성 가능

---

## 🖥️ UI / UX 특징

- 대회 → 클럽 → 팀 순의 명확한 정보 구조
- 스크롤이 많은 환경에서도 탐색이 쉬운 인터페이스
- 실사용(대회 운영, 일정 조율)을 고려한 정보 밀도 설계
